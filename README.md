Домашний проект. Сервис в разработке. Текущая версия разрабатывается как OpenSource проект.

## Roadmap

1.Создание запуск dev версии для деплоя кода в процессе разработки

2.Написание парсера целевой аудитории в твиттер

3.Добавление промоаккаунтов и запуск предварительного массфолловинга по рассписанию

3.Создание скелета frontend'а для юзера, который можно использовать для быстрого запуска

4.Описание принципа работы и правил бесплатного использования для пользователя.

5.Создание таблиц токенов и истории подписок

6.Запуск базового функционала FollowMachine с упором на backend

7.Активация промоаккаунтов

8.Тестирование нагрузок, оптимизация нагрузок, экспериметы с продвижением, геолокацией и мультязычностью, дизайн
