Домашний проект. Сервис в разработке. Текущая версия разрабатывается как OpenSource проект.

## Roadmap (исходя из опыта начала 2013 года)

1.Создание запуск dev версии для деплоя кода в процессе разработки

2.Написание парсера целевой аудитории в твиттер

3.Добавление промоаккаунтов и запуск предварительного массфолловинга по рассписанию

4.Создание скелета frontend'а для юзера, который можно использовать для быстрого запуска

5.Описание принципа работы и правил бесплатного использования для пользователя.

6.Создание таблиц токенов и истории подписок

7.Запуск базового функционала FollowMachine с упором на backend

8.Активация промоаккаунтов

...

M.Тестирование нагрузок, оптимизация нагрузок, экспериметы с продвижением, геолокацией и мультязычностью, дизайн

...

N.Изучение моментов децентрализации проекта
